Лабораторная работа № 3 «Регулярные выражения. Работа с Git»

Технологии: регулярные выражения (C++, Python), системы контроля версий.

Инструменты: QT, PyCharm, git.

Лабораторная работа № 3. Регулярные выражения. Работа с Git.

1) Выполнить задание на тему регулярного выражения (предложив свой вариант или по вариантам).

2) Для хранения кода используйте git (например github.com или bitbucket.org).

Полезные ссылки: интерактивный тренажёр https://learngitbranching.js.org/, книга https://git-scm.com/book/ru/v2/. 


Варианты (балл не выше 34):

1. Слова в алфавите {a, b}, такие, что на третьем месте от начала слова стоит буква а, а на пятом месте с конца – буква b.

2. Слова в алфавите {a, b}, в которых число букв четно.

3. Слова в алфавите {a, b}, не содержащие подстроки ab.

4. Слова в алфавите {a, b}, не содержащие подстроки aab.

5. Слова в алфавите {a, b, c}, в которых нет двух соседних букв b.


Варианты (балл не выше 44):

6. Слова в алфавите {a, b, c}, содержащие подслово вида bxa, где x – произвольная буква алфавита.

7. Слова в алфавите {a, b, c}, в которых за буквой а обязательно следует буква с.

8. Выведите строки, содержащие две буквы "z﻿", между которыми ровно три символа.

9. Выведите строки, содержащие слово, состоящее из двух одинаковых частей (тандемный повтор).

10. Проверить, что строка содержит синтаксически корректный IP-адрес (IPv4).


Варианты (балл не выше 54):

11. Из документа HTML вывести все e-mail.

12. Из документа HTML вывести имена файлов, которые указаны в ссылках на другие документы.

13. Из документа HTML вывести все гиперссылки.

14. Выведите строки, содержащие двоичную запись числа, кратного 3.

15. Проверить, что строка является синтаксически корректным математическим выражением. Математическое выражение может

содержать числа, односимвольные переменные, знаки математических операций, круглые скобки.

Корректные примеры: 17*4+(x-54/(2+4))=y, 2+2, 18-41*с.

Некорректные примеры: +45, 17+4*, (34+1, 45-3), (4+5)).
